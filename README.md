## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/makingdevs/do-hello-world-neodevelop** and import

## Edit this project ![Build status badge](https://github.com/makingdevs/do-hello-world-neodevelop/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/makingdevs/do-hello-world-neodevelop** and click import
